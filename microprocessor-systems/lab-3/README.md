# Лабораторная работа №3 — Память и Регистры

## Цель работы
Реализовать два типа хранилища данных: быстрый доступ внутри ядра (Регистры) и долговременное хранение программы (IMEM).

---

## 🏗 Архитектура памяти

### 1. Регистровый файл (RF)
Это массив из 32 ячеек по 32 бита. У него 3 порта:
* **Порт чтения 1 (асинхронный):** `ra1` -> `rd1`.
* **Порт чтения 2 (асинхронный):** `ra2` -> `rd2`.
* **Порт записи (синхронный):** `wa`, `wd`, `we`. Запись происходит только по `posedge clk`.

### 2. Память инструкций (IMEM)
Это **ROM** (память только для чтения). Она инициализируется при старте из файла `program.mem` командой `$readmemh`.
