# Лабораторная работа №2 — АЛУ (Arithmetic Logic Unit)

## Цель работы
Создать "калькулятор" процессора, который умеет выполнять 16 различных операций (арифметика, логика, сдвиги, сравнения).

---

## 🏗 Схема: Большой Мультиплексор
АЛУ — это **комбинационная схема**. Физически внутри АЛУ все блоки (сумматор, сдвигатель) работают **параллельно** и всегда вычисляют что-то. 
Вход `alu_op_i` управляет мультиплексором на выходе, который просто пропускает нужный результат наружу.

### Два выхода — две задачи
1. **result_o (32 бита):** Сюда идет результат вычислений (1+1=2).
2. **flag_o (1 бит):** Сюда идет результат сравнения (правда/ложь). Используется для команд типа `beq` (Branch if Equal).
