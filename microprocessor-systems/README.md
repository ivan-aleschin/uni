# Микропроцессорные средства и системы — Лабораторные работы

## О проекте
Этот репозиторий содержит выполненные лабораторные работы по курсу «Архитектуры процессорных систем / Микропроцессорные средства и системы». 
Лабы выполняются на языке описания аппаратуры **SystemVerilog** и проверяются в симуляции, а затем (по требованию) синтезируются и прошиваются в **ПЛИС (FPGA)** через **Vivado**.

Официальные материалы курса:
- Репозиторий APS: https://github.com/MPSU/APS  
- Папка с лабораторными работами: https://github.com/MPSU/APS/tree/master/Labs

---

## Содержание (локальные лабы)

1. **ЛР1 — Сумматор (Adder)**  
   Папка: `lab-1/`

2. **ЛР2 — Арифметико-логическое устройство (ALU)**  
   Папка: `lab-2/`

3. **ЛР3 — Регистровый файл и память инструкций**  
   Папка: `lab-3/`

4. **ЛР4 — Простейшее программируемое устройство (CYBERcobra)**  
   Папка: `lab-4/`

---

## Как работать с проектом

### 1) Симуляция дома
Используется:
- **Icarus Verilog** (`iverilog`)
- **GTKWave** (просмотр VCD)

Команды для запуска находятся в `README.md` каждой лаборатории.

### 2) Проверка в Vivado
Используется:
- **Vivado** (синтез, имплементация, прошивка ПЛИС)
- Testbench для симуляции

Пошаговые инструкции также описаны в `README.md` каждой лаборатории.

---

## Полезные файлы
В каждой лаборатории есть:
- `README.md` — практические шаги
- `THEORY.md` — теоретическая база, синтаксис SystemVerilog и пояснения
