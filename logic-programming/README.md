## Запуск SWI-Prolog на NixOS

### Запустить файл и выполнить тесты:

```bash
swipl lab-1.pl
```

После загрузки в интерактивной оболочке набери:

```prolog
?- run_tests.
```

### Или сразу в одну команду (без интерактива):

```bash
swipl -g run_tests -t halt lab-1.pl
```

### Если `swipl` не найден (nix-shell на лету):

```bash
nix-shell -p swi-prolog --run "swipl lab-1.pl"
```
