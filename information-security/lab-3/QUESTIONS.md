# Контрольные вопросы по лабораторной работе №3 (Сети Фейстеля)

---

## 1. Какова структура классической сети Фейстеля?

Классическая сеть Фейстеля имеет **две ветви** (левую и правую), полученные из одного n-битного блока. Каждый раунд выполняет:
1. Вычисление раундовой функции **F** над одной ветвью с параметром **Vᵢ** (зависит от ключа).
2. Наложение результата на другую ветвь с помощью XOR.
3. Перестановку ветвей местами.

Схематически:  
`Lᵢ = Rᵢ₋₁`  
`Rᵢ = Lᵢ₋₁ ⊕ F(Rᵢ₋₁, Vᵢ)`

---

## 2. Что называется раундом в сети Фейстеля?

**Раунд** — это один цикл преобразования, включающий:
- вычисление функции **F** с параметром **Vᵢ**,
- наложение результата на противоположную ветвь (обычно XOR),
- перестановку (обмен) ветвей.

После K раундов формируется итоговый шифрблок.

---

## 3. Какими свойствами обладает сеть Фейстеля?

Основные свойства:

1. **Обратимость**  
   Шифр всегда обратим, даже если функция **F** необратима. Для дешифрования не требуется вычислять **F⁻¹**.

2. **Симметричность процесса**  
   Шифрование и дешифрование выполняются одной и той же схемой, меняется только порядок параметров **Vᵢ**.

3. **Независимость от четности числа раундов**  
   Алгоритм корректно работает при любом количестве раундов (четном или нечетном).

---

## 4. Каким образом используется материал ключа при шифровании?

Ключ используется для формирования раундовых параметров **Vᵢ**.  
В лабораторной работе (вариант 1) ключ **K** (64 бита) делится на две части:
- **K1** — старшие 32 бита,
- **K2** — младшие 32 бита.

Далее для каждого раунда вычисляется:

`Vᵢ = ROL(K1, i) + ROR(K2, i)`

Где:
- **ROL** — циклический сдвиг влево,
- **ROR** — циклический сдвиг вправо.

Параметр **Vᵢ** подается в функцию **F**, поэтому ключ влияет на каждый раунд и на итоговый результат шифрования.
