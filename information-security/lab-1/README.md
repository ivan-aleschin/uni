## 1. Описание используемого метода

В данной лабораторной работе реализован **метод моноалфавитной подстановки**. 
Это один из простейших видов криптографических преобразований, заключающийся в замене символов исходного текста на другие символы того же алфавита по определенному правилу (ключу).

Для успешной реализации этого метода размер множества символов открытого текста и шифротекста должен совпадать. В качестве ключа используется перестановка символов английского алфавита. При шифровании каждая буква исходного сообщения заменяется на соответствующую букву из ключа (подстановочного алфавита). При дешифровании выполняется обратная операция: буква из криптограммы ищется в ключе, и заменяется на соответствующую букву из стандартного алфавита.

Несмотря на простоту реализации, моноалфавитные шифры подстановки обладают слабой криптостойкостью и легко взламываются с помощью частотного анализа.

## 2. Описание исходных данных

- **Исходный алфавит:** Стандартный английский алфавит (26 букв, `ABCDEFGHIJKLMNOPQRSTUVWXYZ`).
- **Подстановочный алфавит (Ключ):** Произвольная перестановка английского алфавита (в реализации используется `QWERTYUIOPASDFGHJKLZXCVBNM`).
- **Открытый текст:** Произвольный текст на английском языке (например, `Information Security is crucial. This is Lab 1 for Substitution Cipher!`).
- **Формат данных:** Данные считываются из текстового файла `input.txt`. Результаты шифрования сохраняются в `encrypted.txt`, а результаты дешифрования — в `decrypted.txt`.

## 3. Алгоритм работы программы

1. **Инициализация:** Создается класс `SubstitutionCipher`, принимающий на вход ключ (перестановку алфавита). Генерируются таблицы преобразования (`encode_map` и `decode_map`) для верхнего и нижнего регистров с использованием функции `str.maketrans`.
2. **Чтение:** Программа считывает открытый текст из файла `input.txt`. Если файл отсутствует, он создается автоматически с тестовым содержимым.
3. **Шифрование:** Исходный текст посимвольно переводится в криптограмму на основе таблицы `encode_map`. Символы, не входящие в английский алфавит (знаки препинания, пробелы, цифры), остаются без изменений.
4. **Сохранение криптограммы:** Зашифрованный текст записывается в файл `encrypted.txt` и выводится на экран.
5. **Дешифрование:** Криптограмма считывается из файла `encrypted.txt` и переводится обратно в исходный текст на основе обратной таблицы `decode_map`.
6. **Сохранение и проверка:** Расшифрованный текст записывается в `decrypted.txt`. Программа сравнивает его с изначальным текстом и выводит сообщение об успешном завершении операции.

## 4. Текст программы

```python
import os
import string


class SubstitutionCipher:
    """
    Класс для шифрования и дешифрования текста методом моноалфавитной подстановки.
    Использует английский алфавит согласно Варианту 1.
    """

    def __init__(self, key: str):
        self.alphabet = string.ascii_uppercase
        self.key = key.upper()

        if len(self.alphabet) != len(self.key) or set(self.alphabet) != set(self.key):
            raise ValueError(
                "Ключ должен содержать все 26 букв английского алфавита без повторений."
            )

        # Создаем таблицы перевода для верхнего и нижнего регистра
        self.encode_map = str.maketrans(
            self.alphabet + self.alphabet.lower(), self.key + self.key.lower()
        )
        self.decode_map = str.maketrans(
            self.key + self.key.lower(), self.alphabet + self.alphabet.lower()
        )

    def encrypt(self, text: str) -> str:
        """Шифрует исходный текст."""
        return text.translate(self.encode_map)

    def decrypt(self, text: str) -> str:
        """Дешифрует криптограмму обратно в исходный текст."""
        return text.translate(self.decode_map)


def main():
    # Ключ - перестановка английского алфавита
    key = "QWERTYUIOPASDFGHJKLZXCVBNM"
    cipher = SubstitutionCipher(key)

    input_filename = "input.txt"
    output_encrypted = "encrypted.txt"
    output_decrypted = "decrypted.txt"

    # Если файла нет, создадим его с тестовыми данными
    if not os.path.exists(input_filename):
        with open(input_filename, "w", encoding="utf-8") as f:
            f.write(
                "Information Security is crucial. This is Lab 1 for Substitution Cipher!"
            )

    # 1. Чтение исходных данных
    with open(input_filename, "r", encoding="utf-8") as f:
        original_text = f.read()

    print("=== ИСХОДНЫЙ ТЕКСТ ===")
    print(original_text)
    print("======================\n")

    # 2. Шифрование
    encrypted_text = cipher.encrypt(original_text)
    with open(output_encrypted, "w", encoding="utf-8") as f:
        f.write(encrypted_text)

    print("=== ЗАШИФРОВАННЫЙ ТЕКСТ (КРИПТОГРАММА) ===")
    print(encrypted_text)
    print("==========================================\n")

    # 3. Дешифрование
    with open(output_encrypted, "r", encoding="utf-8") as f:
        text_to_decrypt = f.read()

    decrypted_text = cipher.decrypt(text_to_decrypt)
    with open(output_decrypted, "w", encoding="utf-8") as f:
        f.write(decrypted_text)

    print("=== РАСШИФРОВАННЫЙ ТЕКСТ ===")
    print(decrypted_text)
    print("============================\n")

    # Проверка на совпадение
    if original_text == decrypted_text:
        print("[УСПЕХ] Исходный текст и расшифрованный текст полностью совпадают.")
    else:
        print("[ОШИБКА] Дешифрование прошло некорректно.")


if __name__ == "__main__":
    main()
```
