# REQUESTS:
# username<TAB>positions
#   - positions — список запрошенных позиций через запятую (1-based, по символам)
#   - предназначено для схемы «выборка символов» (вариант 1)

# Примечание для реализации:
# - Позиции считаются в терминах символов (Unicode codepoints). Для упрощения можно работать с .encode('utf-8')
#   или с Python-строкой (позиции по символам) — уточните в постановке задачи.
# - В отчёте показать, что система корректно обрабатывает пробелы, спецсимволы и символы Юникода.
#
# Примеры ниже — тестовые данные для лабораторной работы.
# ----
