# Контрольные вопросы по лабораторной работе №4 (RSA)

---

## 1. Что такое однонаправленные функции?

Однонаправленная функция — это функция f(x), для которой:
- прямое вычисление значения y = f(x) выполняется эффективно;
- обратное восстановление x по известному y вычислительно сложно.

В криптографии сложность обратной операции используется как основа безопасности.

---

## 2. Основные свойства однонаправленных функций с потайным ходом?

Функции с потайным ходом (trapdoor one-way functions) обладают свойствами:

1. **Легкость прямого вычисления**  
   f(x) вычисляется эффективно.

2. **Сложность обратного вычисления без секрета**  
   Восстановить x по y без дополнительной информации практически невозможно.

3. **Наличие потайной информации (trapdoor)**  
   При знании секретного параметра обратное вычисление становится эффективным.

В RSA потайной информацией являются простые множители p и q.

---

## 3. Какие криптоалгоритмы используют однонаправленные функции?

Однонаправленные функции используются в:
- **RSA** (основана на факторизации n = p⋅q),
- **ElGamal** и **Diffie–Hellman** (основаны на дискретном логарифмировании),
- схемах цифровой подписи (DSA, ECDSA),
- некоторых системах аутентификации и распределения ключей.

---

## 4. Как реализуется программное возведение в степень для больших чисел?

Эффективная реализация строится на методе **быстрого возведения в степень (exponentiation by squaring)**:

- степень разбивается на двоичное представление;
- на каждом шаге выполняется операция умножения и возведения в квадрат по модулю;
- сложность O(log e) вместо O(e).

В Python это реализуется встроенной функцией:

- pow(a, e, n) — быстрое модульное возведение в степень.
