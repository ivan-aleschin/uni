### 1. Метод Кэннона (Cannon's Algorithm)

**Принцип работы:**
- Матрицы разбиваются на блоки размером (N/p) × (N/p)
- Блоки матрицы A сдвигаются влево, блоки матрицы B - вверх
- На каждой итерации происходит перемножение текущих блоков
- После умножения блоки циклически сдвигаются

**Преимущества:**
- Хорошая масштабируемость
- Равномерное распределение нагрузки
- Минимизация коммуникаций

**Недостатки:**
- Сложность реализации
- Требует точной синхронизации

---
### 2. Ленточный метод (Tape Method)

**Принцип работы:**
- Матрица A разбивается на горизонтальные полосы (строки)
- Матрица B разбивается на вертикальные полосы (столбцы)
- Каждый процесс получает одну полосу A и одну полосу B
- Данные циклически сдвигаются между процессами

**Преимущества:**
- Проще в реализации чем метод Кэннона
- Меньше накладных расходов на коммуникации
- Хорошая производительность для больших матриц

**Недостатки:**
- Менее сбалансированная нагрузка
- Ограниченная масштабируемость

---
### 3. Сравнение производительности

**Метод Кэннона:**
- Лучше для большого количества процессов
- Оптимален для систем с топологией сетки
- Время выполнения: O(N³/p + N²)

**Ленточный метод:**
- Лучше для малого и среднего количества процессов
- Проще в реализации и отладке
- Время выполнения: O(N³/p)

---
### 4. Алгоритм Odd-Even Sort для матриц

**Принцип работы:**
- Каждая строка матрицы сортируется независимо
- Используется алгоритм чет-нечетной сортировки
- Параллельная обработка строк

**Сложность:**
- Временная сложность: O(N² log N)
- Параллельная сложность: O(N log N) при p = N

---
### 6. Практические рекомендации

**Для метода Кэннона:**
- Используйте когда p² ≤ N
- Оптимально для квадратных сеток процессов
- Требует аккуратной реализации сдвигов

**Для ленточного метода:**
- Используйте когда p ≤ N
- Проще в отладке
- Лучшая производительность на малых p

**Оптимизации:**
- Локализация данных
- Векторизация вычислений
- Оптимальный выбор размера блока

### 7. Результаты тестирования

Ожидаемые результаты при правильной реализации:
- Ленточный метод быстрее при p < 8
- Метод Кэннона выигрывает при больших p
- Ускорение близко к линейному при оптимальных условиях
