## **Слайд 5: Супер-идея**

**На слайде:**
```
В чём уникальность? 

RAG + LLM = Персонализация нового уровня

• RAG — понимает контекст запроса
• LLM — учится на ваших предпочтениях
```

**Визуал:** Схема: RAG + RecSys → Персонализированный маршрут

**Текст для выступления:**
> В чём наша супер-идея?  Мы комбинируем две технологии, которые обычно используются отдельно. RAG — Retrieval-Augmented Generation — позволяет системе понимать контекст вашего запроса и находить релевантную информацию. LLM — рекомендательная система — учится на ваших предпочтениях и улучшает рекомендации со временем. Эта комбинация даёт уровень персонализации, которого нет у конкурентов.
