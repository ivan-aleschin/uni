## Что реализует код?

Это Flask-приложение с двумя основными направлениями:

### 1. Работа с "почтой"

- Хранит письма в JSON-файле `maillist.json`.
    
- Позволяет:
    
    - получать список писем (с фильтрами по теме и id),
        
    - добавлять новые письма (с вложенными файлами),
        
    - удалять письма.
        
- Файлы вложений загружаются в папку `uploads/` локально.
    

### 2. Интеграция с Яндекс.Диском через API

- Загружает файлы на Яндекс.Диск (`/setFiles`),
    
- Получает список файлов и папок (`/getFiles`, `/getFolders`),
    
- Создаёт папки (`/setFolder`),
    
- Скачивает файлы и упаковывает их в ZIP архив для загрузки (`/downloadAsZip`).
    

Для работы с Яндекс.Диском используются стандартные REST API по адресу:  
`https://cloud-api.yandex.net/v1/disk` с OAuth токеном авторизации.

---

## Где и как изменить ссылки под свои учётные записи?

### 1. OAuth токен Яндекс.Диска

В начале кода:

```python
YA_ACCESS_TOKEN = ""  # <- сюда нужно вставить свой OAuth токен
YA_API_URL = "https://cloud-api.yandex.net/v1/disk"
YA_HEADERS = {
    "Authorization": f"OAuth {YA_ACCESS_TOKEN}",
    "Accept": "application/json",
    "Content-Type": "application/json"
}
```

**Что сделать?**

- Получи свой OAuth-токен Яндекс.Диска ([https://cloud.yandex.ru/docs/oauth/](https://cloud.yandex.ru/docs/oauth/)).
    
- Вставь его в переменную `YA_ACCESS_TOKEN`, например:
    

```python
YA_ACCESS_TOKEN = "AQAAAAA..."  # твой токен
```

- Теперь все запросы к API Яндекс.Диска будут авторизованы от твоей учётной записи.
    

---

### 2. Пути и ссылки на файлы в API

- Все операции с файлами (загрузка, получение, удаление) идут через API с адресом `YA_API_URL`.
    
- Параметры `path` — это путь на твоём диске в Яндекс.Диске, например `/myfolder/myfile.txt`.
    
- Ты можешь изменить в запросах путь `path` на нужный, чтобы работать с нужной папкой.
    

---

### 3. Дополнительные настройки

- Если хочешь использовать другие облачные хранилища (Google Drive, Dropbox), нужно будет:
    
    - Изменить API URL и запросы под их API.
        
    - Подключить авторизацию для этих сервисов.
        
- Для Яндекс.Диска код уже готов, достаточно указать свой токен и пути.

## Итог:

- Для **Яндекс.Диска**:
    
    - Получи OAuth токен.
        
    - Вставь в `YA_ACCESS_TOKEN` в `app.py`.
        
    - Используй API пути (`path`) для нужных папок и файлов.
        
- Для **почты**:
    
    - Письма хранятся локально в `maillist.json`.
        
    - Файлы сохраняются локально в папку `uploads/`.
