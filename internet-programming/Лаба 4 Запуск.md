### 🔄 **Порядок запуска:**

1. **Запусти `server.py`**  
    Этот сервер:
    
    - запускает Flask-приложение;
        
    - отслеживает изменения в файле `templates/index.html` (через `watchdog`);
        
    - при изменении шаблона отправляет событие `reload` в браузер, чтобы автоматически перезагрузить страницу и показать результат атаки.
        
    
    ```bash
    python server.py
    ```
    
2. **Открой в браузере страницу**  
    Перейди по адресу:  
    [http://localhost:5000](http://localhost:5000/)  
    Ты увидишь изначальную «чистую» версию сайта.
    
3. **Запусти `backup.py`**  
    Он делает резервную копию текущего `index.html`, чтобы ты мог восстановить систему после атаки:
    
    ```bash
    python backup.py
    ```
    
4. **Запусти `attack.py`**  
    Этот скрипт:
    
    - удаляет изображения;
        
    - меняет цветовую схему на красно-чёрную;
        
    - подменяет ссылки на вредоносный сайт;
        
    - добавляет скрипт, который включает предупреждение.
        
    
    ```bash
    python attack.py
    ```
    
    После выполнения ты увидишь:
    
    - автоматическое обновление страницы в браузере (из-за `socketio.emit("reload")`);
        
    - отображение _«⚠️ ВНИМАНИЕ! Обнаружена активность угроз!»_;
        
    - ссылки ведут на `http://malicious-site.com`;
        
    - фон стал чёрным, основной цвет — красный;
        
    - изображения исчезли.
        
5. **Для восстановления системы**  
    Запусти `recover.py`, чтобы вернуть оригинальный HTML-шаблон:
    
    ```bash
    python recover.py
    ```

### 💡 Советы

- Можешь логировать `socket.emit("reload")` на стороне клиента, чтобы фиксировать изменения в консоли.
    
- Сделай снимки экрана до/после, чтобы продемонстрировать результат.
    
- Включи сохранённый `backups/backup_*.html` в отчёт — как подтверждение, что ты умеешь делать резервную копию.
