# Лабораторная работа 4

## Задание
**Подготовить 10 осмысленных запросов, демонстрирующих изменения структуры БД (из лабораторной работы 2). Запросы должны включать не менее 3-х таблиц. Обязательны запросы с подзапросами, группировкой и оконными функциями.**

В этой лабораторной работе представлены 10 файлов `01.sql` - `10.sql`, каждый из которых содержит сложный SQL-запрос, полностью соответствующий требованиям:
1. Затрагивает новые колонки/таблицы (ВВП, заблокированные кресла, перенаправленные рейсы).
2. Использует `JOIN` с минимум 3 таблицами.
3. Содержит подзапросы (в `SELECT`, `FROM`, `WHERE` или `HAVING`).
4. Применяет группировку (`GROUP BY`).
5. Использует различные оконные функции (`RANK`, `DENSE_RANK`, `ROW_NUMBER`, `SUM() OVER()`, `COUNT() OVER()`, `LAG`).

Также добавлен файл `00_seed_data.sql`, который добавляет в БД немного тестовых данных (событий), чтобы эти 10 сложных аналитических запросов выдавали не пустые таблицы, а красивые и осмысленные результаты.
